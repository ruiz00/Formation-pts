# Formation Web

Bienvenue dans ce dépôt dédié à la **formation au développement web (frontend)**.
Ce projet contient les ressources, exemples de code et exercices permettant d'apprendre les bases du développement web moderne.

## Objectifs de la formation

Cette formation a pour but de :

* Comprendre le fonctionnement du **web et d'Internet**
* Apprendre les bases du **HTML5**
* Découvrir la mise en forme avec **CSS3**
* Comprendre les bases de **JavaScript**
* Créer des **pages web structurées et modernes**
* Publier un projet sur **GitHub**

## Contenu du dépôt

Le dépôt est organisé en plusieurs sections :

* **HTML** : structure des pages web et balises sémantiques
* **CSS** : mise en forme, couleurs, layout (flexbox, grid)
* **JavaScript** : interactions et scripts côté client
* **Projets pratiques** : mini-projets pour appliquer les connaissances

## Objectif final

À la fin de cette formation, l'apprenant sera capable de :

* Créer un **site web simple**
* Structurer une page avec **HTML**
* Styliser une interface avec **CSS**
* Ajouter de l’interactivité avec **JavaScript**
* Utiliser **Git et GitHub** pour gérer ses projets
